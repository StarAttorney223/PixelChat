PixelChat is a real-time multiplayer chat application built using WebSockets.
It features a retro pixel-style UI and enables users to join rooms, send messages instantly, and share media.

✨ Features
💬 Real-time messaging
👥 Join/Create chat rooms
⌨️ Typing indicator
🖼️ Image & video sharing
🟢 Online users list
🔄 Auto-reconnect on disconnect
🎮 Retro pixel-themed UI
